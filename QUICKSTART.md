# Quick Start Checklist

Complete these steps in order:

## Before You Start
- [ ] Download and install Node.js from https://nodejs.org/
- [ ] Download and install Git from https://git-scm.com/
- [ ] Download and install GitHub Desktop from https://desktop.github.com/
- [ ] Restart your computer

## Verify Installation
Open Terminal/Command Prompt and check:
- [ ] `node --version` shows a version number
- [ ] `npm --version` shows a version number
- [ ] `git --version` shows a version number

## Upload to GitHub (5 minutes)
- [ ] Open GitHub Desktop
- [ ] File → Add Local Repository → Choose this folder
- [ ] Click "Create a Repository" 
- [ ] Click "Publish repository"
- [ ] Uncheck "Keep this code private"
- [ ] Click "Publish Repository"

## Deploy to Vercel (5 minutes)
- [ ] Go to https://vercel.com/signup
- [ ] Click "Continue with GitHub"
- [ ] Click "Add New..." → "Project"
- [ ] Find and import "sceneclips"
- [ ] Click "Deploy"
- [ ] Wait 2-3 minutes
- [ ] Click "Visit" when it says "Congratulations!"

## Test It
- [ ] Open site on your phone
- [ ] Search for a clip
- [ ] Click "Copy Link"
- [ ] Text it to yourself
- [ ] Verify it plays at the right timestamp

## Share It!
- [ ] Text the link to 10 friends
- [ ] Post on social media
- [ ] Get feedback
- [ ] Add requested clips

---

**Total Time: ~30 minutes**

If you get stuck at any step, open README.md for detailed instructions.
